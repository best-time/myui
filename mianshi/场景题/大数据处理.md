## js超过Number最大值

- 大数据的计算
- 格式展示
- 用户输入

背景

- 金融
- 科学计算
- 数据分析

## 解决方案

- BigInt

```javascript
const bigNum = BigInt(9007199254740993) + BigInt(9007199254740993)
```

- decimal.js

```javascript
const Decimal = require('decimal.js')
const decimal = new Decimal("1e+308")
```

- big.js

```javascript
const Big = require('big.js')
const big = new Big("1e+308")
```
